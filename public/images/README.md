# Image Upload Guide for DJ Vish Website

## Required Images

### 1. DJ Hands Background Image
**File:** `dj-hands-background.jpg`
**Location:** `public/images/dj-hands-background.jpg`
**Description:** Black and white close-up photo of DJ hands manipulating professional DJ controller/mixer
**Requirements:**
- Format: JPG or PNG
- Size: Minimum 1920x1080px (Full HD)
- Recommended: 2560x1440px or higher for crisp display
- Quality: High resolution for professional appearance
- File size: Optimized (under 2MB for web performance)

**Usage:** This image is used as the main background in the hero section and as a subtle background in other sections.

## Upload Instructions

1. **Replace the placeholder files** in each directory with your actual images
2. **Use descriptive filenames** (e.g., `wedding-reception-1.jpg`)
3. **Optimize images** for web (compress without losing quality)
4. **Maintain consistent aspect ratios** for gallery displays
5. **Use high-quality images** that represent your professional services

## Image Optimization Tips

- Use tools like TinyPNG or ImageOptim to compress images
- Save images in WebP format when possible for better performance
- Maintain aspect ratios for consistent gallery layouts
- Use descriptive alt text for accessibility
